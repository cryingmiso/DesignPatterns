# Factory pattern
팩토리 패턴(Factory pattern)은 클라이언트로 부터 인스턴스를 생성하는 책임을 갖는 패턴이다.

### 상세설명
 - 추상 팩토리 패턴과 팩토리 메서드 패턴으로 나뉜다.
 - 추상 팩토리 패턴은 팩토리 메서드를 제어하기 위해 사용한다.
 - 추상 팩토리
